What is UltraChrome?

UltraChrome is an open-source, NASA backed, web browser like Google Chrome, but it is more AWESOME. In short, it is superior to chrome -- By Far. Ultra...Chrome.

With the latest UltraChrome 1.4, pages load in O(log(n))time, meaning that a web page 2x as large will only take 1 ms extra to load!!!

UltraChrome 1.3 will take 1ms extra for each 1.5x in size.

Our efforts currently lie in the development of the 1.5 version, which will load pages in O(1) time, meaning that no matter how large a web page is, it will not take any extra time to load!
